### Requerimientos ###
Servidor:
- PHP 5.1 o superior (Última versión estable recomendada)
- MySQL 4 o superior

Cliente:
- Un navegador actualizado con capacidad de ejecutar JS/DOM (probado en Firefox 2/3, Internet Explorer 7, Opera 9, Konqueror, Safari)
- Cookies habilitadas


### Instrucciones de instalación ###
	1. Descomprima Collabtive.
	2. Suba todo a su servidor, incluyendo las carpetas "/files" y "/templates_c".
	   (Puede que necesite crear las carpetas "/files" y "/templates_c" de forma manual antes de la instalación)
	3. Asigne permisos de escritura a los siguientes archivos y carpetas:
	- /templates_c
	- /files
	- /config/standard/config.php
	4. Cree una nueva base de datos MySQL para Collabtive  (collation: utf8_general_ci).
	5. Acceda con su navegador a "install.php" y siga las instrucciones.
	6. Cuando la instalación haya terminado correctamente, borre el archivo "install.php" y "update.php".


### Instrucciones para actualizar ###
	1. Descomprima Collabtive.
	2. Descargue el archivo "config.php" de su servidor.
	3. Copie el archivo "config.php" en la carpeta "/config/standard/", reemplazando el que está vacío.
	4. Suba todo a su servidor y reemplace las versiones anteriores de Collabtive.
	5. Acceda con su navegador a "update.php" y siga las instrucciones.
	6. Cuando la instalación haya terminado correctamente, borre el archivo "install.php" y "update.php".
