### Prérequis : ###
Serveur:
- PHP 5.1 ou supérieur (Dernière version stable recommandée)
- MySQL 4 ou supérieur

Client:
- Navigateur récent supportant JS/DOM (testé sur Firefox 2/3, Internet Explorer 7, Opera 9, Konqueror, Safari)
- Cookies activés
- Flash activé (optionel , utilisé pour les diagrammes et les sons)


### Instructions d'installation : ###
1. Décompressez l’archive
2. Uploadez le tout,  en incluant les dossiers vides  /files et /templates_c, sur votre serveur
(Vous devez créer les dossiers /templates_c et /files manuellement avant l'installation)
3. Vous devez permettre l'accès en écriture sur les dossiers et fichiers suivants :
- /templates_c
- /files
- /config/standard/config.php
4. Créez une nouvelle base de données MySQL pour Collabtive
5. Faites pointer votre navigateur sur install.php et suivez les instructions à l'écran


### Instructions de mise à jour : ###
1. Décompressez l’archive Collabtive
2. Récupérez le fichier config.php de votre serveur
3. Déplacez votre fichier config.php dans le dossier dans lequel vous avez décompressé l’archive, en remplacement du fichier vide
4. Uploadez le tout sur votre serveur en écrasant les anciens fichiers
5. Faites pointer votre navigateur sur update.php
