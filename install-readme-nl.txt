### Vereisten ###
Server:
- PHP 5.1 of hoger (aanbevolen om meest recente stabiele versie te gebruiken)
- MySQL 4 of hoger

Client:
- Recente browser met JS/DOM (getest op Firefox 2/3, Internet Explorer 7, Opera 9, Konqueror, Safari)
- Cookies aanzetten


### Instructies voor installatie ###
	1. Bestanden uitpakken.
        2. Alles uploaden, inclusief de lege /files en /templates_c mappen naar je server.
		(optioneel dien je de /templates_c en /files manueel aan te maken.)
	3. Geef schrijfrechten aan volgende mappen en bestanden:
		- /templates_c
		- /files
		- /config/standard/config.php
	4. Maak een nieuwe Mysql database (type MyISAM) voor Collabtive.
	5. Voer http://www.mijndomein.xx/install.php uit en volg de instructies.
	6. Indien de installatie succesvol is, verwijder dan install.php en update.php.


### Update instructies ###
	1. Bestanden uitpakken
	2. Kopieer config.php van jouw server
	3. Vervang de lege config.php in map /config/standard/ door de config.php uit de vorige stap.
	4. Upload alles naar je verver, waarbij je alle bestanden op de server laat overschrijven
	5. Voer http://www.mijndomein.xxupdate.php uit.
	6. Indien de update succesvol is, verwijder dan install.php en update.php.
