### System krav ###
Server:
- PHP 5.1 eller nyere (Seneste stabile version er anbefalet)
- MySQL 4 eller nyere

Klient:
- Nye JS/DOM kompetente browsere (testet på Firefox 2/3, Internet Explorer 7, Opera 9, Konqueror, Safari)
- Cookies skal accepteres


### Installation instrukser ###
    1. Udpak Collabtive arkivet.
    2. Upload alt, inklusiv de tomme /files og /templates_c mapper, til din server.
	   (Eventuelt kan du være nødsaget til at oprette /files og /templates_c manuelt inden installationen.)
	3. Gør følgende mapper og filer skrivbare:
		- /templates_c
		- /files
		- /config/standard/config.php
	4. Opret en ny MySQL database til Collabtive  (collation: utf8_general_ci).
	5. Naviger din browser til install.php og følg de instrukser der bliver givet.
	6. Hvis installationen bliver gennemført, slet da install.php og update.php.


### Updatering instrukser ###
    1. Udpak Collabtive Archive
    2. Hent din config.php fra din server
    3. Put din config.php i mappen /config/standard/, overskriv den tomme fil.
    4. Upload alt til din server, overskriv gamle Collabtive filer
    5. Naviger din browser til update.php.
	6. Hvis updateringen bliver gennemført, slet da install.php og update.php.
